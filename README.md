# Album Maker

Create a HTML album and upload via SFTP to make available on hosting - used during lockdown home schooling to share the kids 
work with the teachers

# Installation

Something like::

  python -m venv venv
  venv\Scripts\activate.bat
  pip install -r requirements.txt

## maker.py

Command line script

## gmaker.py

Simple GUI
